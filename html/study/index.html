단축키

      <div> class="col1"> </div>

용어

  • position : 태그를 어떻게 위치시킬지를 정함
    • absolute : 절대 좌표와 함께 위치 지정
    • relative: 원래 있던 위치를 기준으로 좌표를 지정
    • static : 기본값, 다른 태그와의 관계에 의해 자동으로 배치되며 위치 임의 설정 불가능
    • fixed: 스크롤과 상관없이 항상 문서 최 좌측상단을 기준으로 좌표 고정
    • inherit: 부모 태그의 속성값 상속
    • position을 absolute나 fixed로 설정시 가로 크기가 100%가 되는 block 태그의 특징이 사라짐
  • aria-hidden : 시각적으로 숨겨지지 않는 콘텐츠를 보조 기술에서 제외함
    • true : 보조기술 사용자의 콘텐츠 탐색을 제한(두번 읽히지 않기 위함)
      (ul, table과 같이)하위 요소를 포함할 수 있는 요소에 aria-hidden을 "true"로 설정하면 하위 요소까지 숨길 수 있음
    • false : 접근성 API가 보조기술 사용자에게 숨겨진 콘텐츠를 노출하여 콘텐츠를 탐색할 수 있음
    • vertical-align
      • inline 이나 inline-block 구조에만 적용됨
        따라서 block 구조인 ⁢div>는 이 속성을 썼을 때 적용되지 않음
      • 요소 자체만을 정렬하고, 내용 자체에는 영향을 미치지 않음
      • table cell에 적용 할 때에는 내용에 영향을 미침
      • 같은 줄에서 인라인 요소에 따라 높낮이가 달라질 수 있고, 그 줄에 있는 line-height 설정에 따라서도 달라질 수 있음
  • em : 강조하는 텍스트에 사용. 기본 모양은 기울임꼴
  • strong : 중요한 텍스트에 사용 기본 모양은 굵은 글자

기타

  • 크로스 브라우징(Cross Browsing) : 표준 웹기술을 채용하여 다른 기종 혹은 플랫폼에 따라 달리 구현되는 기술을 비슷하게 만듦
