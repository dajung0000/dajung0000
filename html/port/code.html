웹표준이란?

웹에서의 표준은 W3C의 토론을 통해 나온 권고안(recomendation)을 말하며,

관련 기술로는 구조 HTML, 표현 CSS, 동작Script가 있습니다.

웹접근성이란?

법적 관점에서의 웹 접근성은‘모든 사용자가 신체적·환경적 조건에 관계없이 웹에 접근하여 이용할 수 있도록 보장하는 것’입니다.

여기서 신체적 조건이란, 일반 사용자는 물론, 장애를 가진 사람, 고령자 등을 의미하며,

환경적 조건이란 다양한 기기(PC, Mobile, PDA 등), OS(운영체제), 웹 브라우저(Internet Explorer, FireFox, Safari,Chrome, Opera 등) 또는

저사양 및 저속회선 사용자나 이미지, 동영상 등을 볼 수 없는 환경 등을 의미합니다.

IR기법이란?

IR기법은 이미지 대체텍스트 제공을 위한 CSS기법 입니다.

웹접근성은 몸이 불편한 사람이라도 정보를 이용하는데 불편함이 없도록 하는것이 목적에 있기 때문에

만약 시각 장애인이 사이트를 이용할 경우 텍스트는 리더기로 읽어줄 수 있지만 이미지는 읽을 수가 없기 때문에

이미지에도 텍스트를 붙여 줍니다.

이미지 스프라이트 기법이란?

배경 이미지를 낱개로 제작하지 않고, 비슷한 성격의 배경 이미지를 하나의 파일로 제작 후에

background-position을 이용하여 배치하는 방법입니다.
